VULNERABILITY MAP
=================

VULNERABILITY       LOCATION (INSECURE)                     FIX LOCATION (SECURE)
-----------         -------------------                     ---------------------
Reflected XSS       GET /api/rooms?city=<payload>           bleach.clean + html.escape
                    SearchRooms.js (dangerouslySetInnerHTML)  React text interpolation

Stored XSS          POST /api/bookings (guest_name, sr)     bleach.clean before DB insert
                    POST /api/reviews (content)             React text interpolation in render
                    ViewBooking.js (dangerouslySetInnerHTML)

IDOR                GET  /api/bookings/<id>                 X-Owner-Token header check
                    POST /api/bookings/<id>/cancel          secrets.token_urlsafe per booking
                    POST /api/bookings/<id>/pay

Open Redirect       GET /api/redirect?next=<any-url>        urlparse + ALLOWED_REDIRECT_HOSTS
